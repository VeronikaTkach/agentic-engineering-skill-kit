# Usage Prompts

Use these prompts when invoking this skill in Claude, ChatGPT, Gemini, Cursor, Claude Code, or another agentic coding environment.

## Astro Review

```text
Use the astro-framework-best-practices skill.
Act as a senior Astro developer with 5+ years of experience.
Review this Astro project or implementation for architecture, prerender/SSR/hybrid strategy, islands, hydration, routing, images/media, SEO-safe HTML, performance, Core Web Vitals, and production readiness.
Start with findings by severity, then give a concrete fix plan.
```

## Astro Architecture Choice

```text
Use the astro-framework-best-practices skill.
Help choose the Astro architecture for this project.
Compare static/prerender, hybrid SSR, full SSR, server islands, and framework islands.
Justify the recommendation through simplicity, performance, crawlability, maintainability, and deploy risk.
```

## Astro Implementation Plan

```text
Use the astro-framework-best-practices skill.
Create an implementation plan for this Astro task.
Include files, rendering mode, data flow, hydration directives, image/media strategy, SEO-safe HTML, and verification checklist.
```

## Source-Grounded Answer

```text
Use the astro-framework-best-practices skill and verify current primary sources before answering.
Cite only official sources: Astro Docs, Google Search Central, and web.dev.
Separate facts from documentation and engineering inferences.
```
