# Usage Prompts

Use these prompts when invoking this skill in Claude, ChatGPT, Gemini, Cursor, Claude Code, or another agentic writing/coding environment.

## Music Artist SEO Audit

```text
Use the music-artist-landing-seo skill.
Act as an SEO specialist and technical SEO reviewer with 5+ years of experience.
Review this music artist website for entity SEO, crawlability, metadata, schema, image/video SEO, Core Web Vitals, sitemap, canonical URLs, social previews, and launch readiness.
Start with critical findings, then give recommendations and next steps.
```
